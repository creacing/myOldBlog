js的类型基础

js一共有7种内置类型分别是

null，undefined，boolean，string，
number，object，symbol

用typeof运算符可查看值的类型 返回的是一个字符串

特殊例子

(js遗留下来的一个bug) typeof null = “object”;
另一种情况 typeof function(){} = “function”;
然而function并不是js的内置类型，function也是object(对象)
typeof [1,2] = “object” 数组也是object(对象)

js 中变量是没有类型的，只有值才有类型
